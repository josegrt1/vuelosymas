# Vuelos y Más

*Vuelos y Más* es un proyecto web desarrollado como parte de un curso para aprender a estructurar y maquetar sitios web usando *HTML, CSS avanzado y Bootstrap 5*. El sitio ofrece un sistema ficticio de reserva de vuelos, información de viaje y servicios premium.

## ✈ Estructura del Proyecto

vuelosymas/
├── index.html              # Página principal
├── pages/
│   ├── Buscar.html         # Buscador de vuelos
│   ├── informacion.html    # Información y consejos de viaje
│   ├── pluslive.html       # Beneficios y servicios premium
│   └── Tureserva.html      # Confirmación de reserva
├── css/
│   └── style.css           # Estilos personalizados avanzados
├── img/
│   └── [imágenes usadas]   # Logos y fotos de destinos

## 🚀 Tecnologías Utilizadas
- HTML5 y CSS3
- Bootstrap 5 (via CDN)
- Flexbox y Grid (estructuración avanzada)
- JavaScript (librerías de Bootstrap)

## 🛠 Cómo ejecutar
1. Clona o descarga el repositorio.
2. Abre index.html en tu navegador preferido.
3. Navega por las diferentes páginas para explorar las funcionalidades.
